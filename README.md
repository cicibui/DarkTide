## 目录结构

```
├── DarkTide.exe
├── word.json                    #轮播词
├── tools/
│   ├── ZeroEye.exe              # 白文件扫描工具
│   ├── sgn.exe                  # sgn混淆工具
│   ├── ResourceHacker_Chs.exe   # 资源注入工具
│   ├── subsystem_mod.exe        # PE 子系统修改工具
│   ├── keystone.dll             # sgn 依赖
│   ├── whiteFiles/              # 扫描后收录的白文件
└── output/                      # 免杀生成结果目录
```
